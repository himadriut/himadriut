# Featured AI Projects

## Alzheimer's Detection from Speech (ADReSSo21)

Problem  
Early detection of Alzheimer's disease using spontaneous speech signals.

Dataset  
ADReSSo21 challenge dataset.

Method
- TF-IDF text representation
- Linear SVM classifier
- Logistic Regression baseline
- Repeated 5x5 cross-validation

Results

| Model | Balanced Accuracy |
|------|-------------------|
| Logistic Regression | 0.735 |
| Linear SVM | **0.785** |

Key Insight  
ASR transcript normalization significantly improves classification performance.

## Retrieval-Augmented Generation for Medical Question Answering

Problem  
Large language models hallucinate when answering clinical questions.

Approach  
Build a retrieval-augmented generation pipeline combining semantic search and LLM reasoning.

Pipeline

User Query  
→ Embedding Model  
→ Vector Database (FAISS)  
→ Top-k Document Retrieval  
→ LLM Answer Generation

Evaluation Metrics
- Retrieval precision
- Grounded answer score
- Hallucination rate
